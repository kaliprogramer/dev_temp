| 🧠 1. WHAT YOU ARE DOING                                |
|______________________________________________________________|
|
| 👉 Sending emails from Django backend
| 👉 Used for:
|    - signup verification
|    - password reset
|    - notifications
|
|------------------------------------------------------------|

| ⚙️ 2. BASIC EMAIL SETTINGS (settings.py)                |
|______________________________________________________________|
|
| => EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
| => EMAIL_HOST = 'smtp.gmail.com'
| => EMAIL_PORT = 587
| => EMAIL_USE_TLS = True
| => EMAIL_HOST_USER = 'your_email@gmail.com'
| => EMAIL_HOST_PASSWORD = 'your_app_password'
|
|------------------------------------------------------------|

| ⚠️ IMPORTANT (GMAIL SETUP)                              |
|______________________________________________________________|
|
| 👉 Use APP PASSWORD (not your real password)
| 👉 Enable 2-Step Verification in Google account
|
|------------------------------------------------------------|

| 📤 3. SEND SIMPLE EMAIL (views.py)                      |
|______________________________________________________________|
|
| => from django.core.mail import send_mail
|
| => send_mail(
|      subject='Welcome!',
|      message='Thanks for joining our app',
|      from_email='your_email@gmail.com',
|      recipient_list=['user@gmail.com']
|    )
|
|------------------------------------------------------------|

| 🚀 4. HTML EMAIL (PRO STYLE)                            |
|______________________________________________________________|
|
| => from django.core.mail import EmailMultiAlternatives
|
| => email = EmailMultiAlternatives(
|      subject='Welcome!',
|      body='Plain text version',
|      from_email='your_email@gmail.com',
|      to=['user@gmail.com']
|    )
|
| => email.attach_alternative("<h1>Welcome</h1>", "text/html")
| => email.send()
|
|------------------------------------------------------------|

| 🔐 5. EMAIL FOR VERIFICATION (REAL USE CASE)            |
